## ğŸ’» Cara Install (Local Development)

Ingin mencoba menjalankan proyek ini di mesin lokal Anda? Ikuti langkah berikut:

1.  **Clone Repository**
    ```bash
    git clone [https://github.com/yhyaa294/iikan-by-triby.git](https://github.com/yhyaa294/iikan-by-triby.git)
    cd iikan-by-triby
    ```

2.  **Install Dependencies**
    Pastikan Node.js sudah terinstall.
    ```bash
    npm install
    ```

3.  **Jalankan Development Server**
    ```bash
    npm run dev
    ```

4.  **Akses Aplikasi**
    Buka browser dan kunjungi `http://localhost:5173`
